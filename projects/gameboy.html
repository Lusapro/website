Gameboy

A self-made handheld “Gameboy-style” console with the game Snake. Built as a full prototype from electronics to code and 3D-printed casing.

Arduino Nano C++ Soldering 3D print

What I built

  • Handheld console with buttons and display
  • Snake game (custom code)
  • 3D-printed enclosure and internal mounting
  • Wiring, soldering, and iterative prototyping

Challenges

  • Fitting components neatly inside a compact case
  • Reliable button feel and responsiveness
  • Power and stability during gameplay

Quick facts

Status: Prototype
Category: Embedded + 3D print
Main tech: Arduino Nano, C++
